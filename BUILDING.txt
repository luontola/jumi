Requirements:

  Maven 2        (http://maven.apache.org/)
  Java SE 6 JDK  (http://java.sun.com/javase/downloads/index.jsp)

Run all unit tests:

  mvn clean package

Run also the end-to-end tests:

  mvn clean verify

Gather the distribution files generated by the above commands:

  cp dist/dimdwarf-dist/target/*.zip target/*.zip <where-to-copy>

Other useful commands:

  mvn clean install
  mvn clean deploy
  mvn dependency:tree
  mvn versions:display-dependency-updates
